# 📊 Student Performance Analysis (Excel + Tableau)

## 📌 Overview
This project focuses on analyzing student academic performance using Excel and Tableau. The goal is to identify trends, evaluate performance, and generate insights that can help improve learning outcomes.

---

## 🎯 Objectives
- Analyze overall student performance  
- Identify top-performing and low-performing students  
- Compare subject-wise performance  
- Provide insights for improvement  

---

## 📂 Dataset
**Source:** Sample / student dataset  

**Contains:**
- Student Name / ID  
- Subject  
- Marks / Score  
- Attendance (optional)  
- Class / Section  

---

## 🛠️ Tools & Technologies
- **Excel** – Data cleaning, Pivot Tables, VLOOKUP/XLOOKUP  
- **Tableau** – Dashboard creation and visualization  

---

## 📊 Key Features
- Interactive dashboard with filters  
- Subject-wise performance analysis  
- Student-wise score comparison  
- Overall performance trends  
- Visual charts for easy understanding  

---

## 🔍 Insights Generated
- Identified top-performing students  
- Highlighted subjects where students struggle  
- Found patterns in performance trends  
- Enabled comparison across different students and subjects
